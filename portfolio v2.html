Hero · MCP Quartet One bet, four targets

Model Context Protocol servers for Blender, Houdini, Nuke, and Natron

Four MCP servers that expose each application's Python API as structured tools an LLM can call directly. The Nuke and Houdini builds add a persistent memory layer, CLAUDE.md behavioral rules, BM25-based RAG over a growing best-practices file, and bidirectional events — so the assistant gets smarter the more you use it, instead of starting from scratch each session. blender-mcp currently surfaces 175 tools across 25 subsystems.
